# 🏋️‍♂️ FitPro Landing Page

A modern and responsive landing page for a fitness training studio, built entirely with **HTML** and **CSS**.
This project showcases a clean layout, smooth navigation, and visually engaging design suitable for fitness trainers, gyms, or coaching businesses.

---

## 🚀 Features

* Fully responsive design that adjusts to all screen sizes
* Fixed navigation bar with smooth scrolling links
* Eye-catching hero section with overlay text
* About, Services, Reviews, and Contact sections
* Service and review cards with hover animations
* Simple, professional footer with social media links

---

## 🧩 Technologies Used

* **HTML5** — Page structure and semantic elements
* **CSS3** — Styling, layout, and responsiveness

---

## ⚙️ Installation

1. **Clone or download the project**

   ```bash
   git clone https://github.com/your-username/fitpro-landing-page.git
   ```
2. **Open the project folder** in your code editor (e.g., VS Code).
3. **Run the site** by opening `index.html` in your browser.

No dependencies or build tools required — it’s pure HTML and CSS.

---

## 🧠 Usage

* Replace the sample images (`pexels-leonardho-1552252.jpg`, `pexels-goumbik-669584.jpg`) with your own.
* Update text content, colors, and fonts in `LandingP.css` to match your brand.
* Add links to your actual social media accounts in the footer.
